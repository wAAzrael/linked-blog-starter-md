# Weekly Schedule

## Suggested Structure

### Monday
- Mechanics block
- Ranked
- Short match review

### Tuesday
- Map study
- Ranked
- Update mistake database

### Wednesday
- Aim maintenance
- Ranked
- Review one difficult loss

### Thursday
- Agent and utility work
- Ranked
- Pro VOD notes

### Friday
- Focused ranked session
- No major routine changes

### Saturday
- Longer VOD review
- Weak map practice
- Ranked only if mentally fresh

### Sunday
- Weekly review
- Update next week's focus
- Light practice or rest

## Weekly Minimums

- [ ] 3 focused ranked sessions
- [ ] 1 self-VOD review
- [ ] 1 pro-VOD review
- [ ] 1 map study block
- [ ] 1 weekly review
