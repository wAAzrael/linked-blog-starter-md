# Crosshair Library

> Imported from the previous Obsidian vault. Every original crosshair code has been preserved.

## How to Import a Crosshair

1. Open **VALORANT Settings**.
2. Go to **Crosshair**.
3. Select **Import Profile Code**.
4. Copy one complete code from below.
5. Test it in the Range or Deathmatch before using it in ranked.

## Selection Rule

Use a crosshair that remains readable during movement and gunfights. Avoid changing it after every bad match; the crosshair is rarely the real culprit. Tiny placebo goblin, huge confidence.
