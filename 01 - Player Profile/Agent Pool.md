# Agent Pool

## Core Pool

### Omen
Best for flexible, proactive controller play.

### Astra
Best for structured teamplay, global utility, and round control.

### Clove
Best when playing aggressively while still filling controller responsibilities.

## Agent Selection Rule

Choose the agent based on:
1. Map strength
2. Team composition
3. Your current confidence
4. The role your team actually needs

Do not select only based on wanting more kills.
