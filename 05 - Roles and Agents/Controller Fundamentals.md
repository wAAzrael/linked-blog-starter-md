# Controller Fundamentals

## Core Responsibilities

- Deny dangerous sightlines
- Help teammates take space
- Preserve utility for mid and late round
- Communicate smoke timing
- Survive long enough to create a second utility cycle
- Understand when the controller must become proactive

## Controller Death Test

Before taking a risky fight, ask:
1. Has my important utility already been used?
2. Can a teammate trade me?
3. Does winning this duel materially change the round?
4. Is there a safer way to create the same value?
