# KovaaK's Playlist Library

> Imported and reorganized from the previous vault.

## How to Use This Library

Do not run every playlist at once. Pick the playlist that matches the weakness identified in [[Recurring Mistakes]] or a VOD review.

### Suggested Decision Tree

- **Unstable first shot:** First Bullet Accuracy, Prime Static, Micro Clicking
- **Slow or inaccurate flicks:** ethFlick, Prime Flicks, Flick Tech
- **Weak mouse control:** ethSmooth, ADVANCED SECTION Control
- **Reaction-speed focus:** Reflex, SRT
- **Plateau in static clicking:** Static Plateau Breaker I or II
- **Operator focus:** OP God
- **Short pre-ranked preparation:** Warmup
