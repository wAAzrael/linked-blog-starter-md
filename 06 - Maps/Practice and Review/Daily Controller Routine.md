# Daily Controller Routine

## 5-Minute Strat Warmup

1. Open one map file.
2. Read Map Identity + Current Focus.
3. Read your agent section.
4. Say out loud:
   - My attack default is...
   - My defense setup is...
   - My retake smoke is...
   - My first mistake to avoid is...

## Ranked In-Game Comms

Use short calls:

- "Default first, don't die."
- "I smoke when we contact."
- "I have one smoke for late."
- "Do not rotate yet, no spike."
- "Group after pick."
- "Play post, no hunt."

## 10-Minute Post Session

- Fill Mistake Tracker.
- Pick one repeated mistake.
- Update confidence score for each map played.
