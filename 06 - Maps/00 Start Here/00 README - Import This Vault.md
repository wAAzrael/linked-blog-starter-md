## How To Use This

1. Open [[01 - Radiant Controller System]] before ranked.
2. Open the map you are queueing into.
3. Pick your agent section inside that map.
4. After the match, fill [[Mistake Tracker]] and [[VOD Review Sheet]].
5. Update confidence scores weekly.

## Weekly Focus System

Use only **one main focus per map** per week. Do not try to fix everything at once.

Example:

- Ascent: stop dying before second smoke.
- Breeze: improve long-range discipline and avoid dry Mid peeks.
- Haven: rotate later on defense.
- Summit: learn wall timing and Mid control.
